Quick Installation

Get up and running in just a few minutes

1

Clone Repository

Get the project from GitHub with all files included

git clone https://github.com/Rishabh1925/scene_localization_system.git cd scene_localization_system
2

Install Dependencies

Install required Python packages for computer vision and machine learning

pip install -r requirements.txt # Or install individually: pip install torch torchvision transformers opencv-python pillow matplotlib numpy scipy
3

Run the Application

Start detecting objects with natural language queries

python scene_localizer.py # Follow the interactive prompts: # 1. Select your image # 2. Enter search queries # 3. View results!

üí° Pro Tip

For best results, use clear, well-lit images and specific queries like "elderly person with hat" instead of just "person"
